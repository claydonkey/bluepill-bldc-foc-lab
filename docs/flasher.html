Flashing Notes

Chrome or Edge on desktop is required. WebUSB is not broadly available elsewhere. This first version assumes a standard DFU-class interface and the current Bluepill app image layout. If automatic leave/reset does not work on your board, unplug and reconnect after the write completes.
